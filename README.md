# universal-encoder-tool
All-in-one Python GUI for encoding
A cross-platform Python/Qt desktop app bundling:
- Text/Data encoders (Base64, URL, HTML)
- Crypto (AES-EAX, SHA-256)
- NN encoders (embeddings, language detection, summarization)
- Media transcoding (MP3, H.264)
- QR Code & Steganography
- File encryption/decryption

## Quickstart

```bash
python -m venv venv
source venv/bin/activate      # Windows: venv\\Scripts\\activate.bat
pip install -r requirements.txt
python main.py
